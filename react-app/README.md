# Card List React App

This project is a simple card list application built with React.  It fetches data from a JSON server and displays it in a visually appealing way.

## Prerequisites

- Node.js and npm (or yarn)


## Setup

1. Clone the repository:
   \`\`\`
   git clone https://github.com/charkhaniakash/Todo-assignment.git
   cd react-app
   \`\`\`

2. Install dependencies:
   \`\`\`
   npm install
   \`\`\`

## Running the application

1. Start the React application:
   \`\`\`
   npm start
   \`\`\`

2. Open your browser and navigate to `http://localhost:3000` to view the application.

## Technologies Used

- React
- axios for API calls
- react-transition-group for animations

## Features

- Displays a list of cards with images and text.
- Uses smooth animations for transitions.
- Fetches data from a JSON server.
